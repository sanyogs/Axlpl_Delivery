Root cause (most Partial failures)

App and admin UI use human-readable codes (bag BAG…, manifest MUM075, trip LH…). Several endpoints only accept numeric IDs. listbags / listmanifests return empty {}, so the app cannot discover numeric IDs. assignlinehaul returns linehaul_id: 0 and only trip_no.

Also missing: branch/hub dropdown API (Delhi, Chennai, etc.) for Hub Scan — not in the 26-endpoint list.
